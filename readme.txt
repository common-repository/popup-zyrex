== Description ==
Pop Up is a WordPress plugin that allows you to create and display notifications on your website in the form of a banner. The plugin allows you to create different types of notifications, such as notifications about news, special offers, invitations to subscribe to a newsletter or other information. The pop up adapts its size to the banner. The plugin is easy to use and can be set up in a few simple steps. Simply upload the banner and activate it. The visitor to our site after closing the popup will not see it for a day.
Welcome to my programming website: <a href="https://zyrex.eu/">zyrex.eu</a>

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through `Plugins` screen in WordPress Admin Panel.

== Frequently Asked Questions ==

= Can I add an expiration date? =

By default, the cookie is deleted after one day.
